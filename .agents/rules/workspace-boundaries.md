# Workspace Boundaries

The coding-agent workspace is the cockpit. Project repos are work sites.

Use this workspace for:

- Cross-project state
- Session logs
- Durable decisions
- Project pointers
- Global coding-agent rules

Use target repos for:

- Product source code
- Project tests
- Project docs
- Project-specific agent instructions
- Project-specific workflows and tools

Never commit secrets, runtime cache, generated media, or temporary scratch artifacts unless Eddie explicitly asks.
