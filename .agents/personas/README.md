# Personas (Who I Am)

This directory defines the operational identities (personas) of the coding agents working within Eddie's Coding Agent Workspace.

## Definition

- **Persona = Who I am**: A persona outlines the mindset, capabilities, behavioral boundaries, and communication style that the agent adopts upon booting. It defines the general role (e.g. senior fullstack developer, research analyst, review auditor) rather than specific technical tutorials or instructions.
