# Handoff Log: 2026-06-19 Session

## Accomplishments
* **Nunito Black Glow Style**: Updated subtitle style to the custom "Nunito Black Glow" dual-layer visual effect.
* **Aesthetic Optimization**:
  - Identified why the text in the 1080p landscape screenshot looked ugly/blobby:
    1. Landscape font size was hardcoded to a tiny `FontSize=18`.
    2. Outlines (`Outline=4` and `Outline=10`) were hardcoded and too thick for size 18, swallowing the letters.
    3. Letters fused together due to thick outlines.
  - Implemented dynamic proportional font size (`font_size = max(1, round(0.034 * height))`) for BOTH portrait and landscape.
  - Implemented proportional outlines: `text_outline` (11% of font size) and `glow_outline` (24% of font size).
  - Added `Spacing=1.5` to separate characters and prevent outline-blob merging.
  - Raised the vertical margin to `MarginV=18` for better safe-area breathing room.
* **Conditional Vocal Separation**: Optimized the pipeline orchestration to skip vocal isolation when Siren TTS credentials are not provided.
* **Pipeline Caching (`--cache-dir` & `--ref-dir` & `--rebuild-step`)**:
  - Implemented intermediate caching of files (`audio.wav`, `audio.zh.srt`, `coords.txt`, `audio.vi.srt`, `bgm.wav`, `tts_voice.wav`, `masked.mp4`) to skip heavy pipeline stages.
  - Added step `mask` to caching so that the heavy OpenCV masking loop is skipped when `masked.mp4` exists in the cache, allowing subtitle styling tweaks to compile in under 3 seconds.
  - Standardized testing by creating dummy `bgm.wav` and `tts_voice.wav` baseline files inside `temp/reference_cache` (copied from `audio.wav`) to allow offline verification of the audio mixing/ducking code.
  - Verified selective rebuilding using `--rebuild-step mask`, which correctly deleted the cached `masked.mp4` and regenerated it using OpenCV while preserving all other cached items.
