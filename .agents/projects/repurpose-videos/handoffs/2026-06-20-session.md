# repurpose-videos Handoff - 2026-06-20

## 1. What was Accomplished
We successfully finalized and verified the layout optimization for **Portrait (9:16) video subtitles**, resolving wrapping issues and vertical centering gaps.

*   **Greedy Wrapping Algorithm**:
    *   Replaced the length-balancing `textwrap` logic with a custom `greedy_wrap` function in `pipeline/video.py`.
    *   Increased the usable text container width to **96%** of the frame (2% margins on each side).
    *   Tuned Bungee font character width coefficient to **0.28** (~37 characters per line at font size 65).
    *   Subtitles now fill the primary line completely before wrapping to the second line, yielding a human-edited look.

*   **Dynamic MarginV Centering (Curing Double-line Offset)**:
    *   Previously, the vertical margin (`MarginV`) was calculated statically based on a single line height, causing 2-line subtitles to be shifted too high and drift out of the center of the masking zone `[y_min, y_max]`.
    *   Implemented **Dynamic MarginV** per subtitle dialogue event. The code now parses the actual line count and font size of each cue, estimates the geometric height of the text block (`(num_lines - 1) * 1.20 * font_size_actual + 0.85 * font_size_actual`), and dynamically shifts the baseline so the text block centers perfectly in the vertical band.
    *   Both single-line and double-line subtitles are now precisely centered inside the mask band.

*   **Project-Level Testing Documentation**:
    *   Wrote a dedicated guide at `docs/testing_and_debugging.md` directly inside the target repository. This outlines the pipeline steps, CLI testing commands, intermediate cache files, and the ASS layout tuning details for future developers or agents.
