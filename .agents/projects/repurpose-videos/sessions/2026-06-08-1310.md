## Context
Integrating the 'Siren' internal TTS engine and AI Vocal Separation (Demucs) into the video repurposing pipeline to produce studio-quality repurposed videos with Vietnamese voiceovers.

## Changes Made
1. **TTS Synthesis & Timeline Alignment (`repurpose.py`)**:
   - Added `parse_srt_time` and `parse_srt_file` to parse Vietnamese SRT subtitles.
   - Added `synthesize_siren_tts` using the `httpx` client to hit the Siren Talks API synchronous `POST /api/audio/synthesize` endpoint, downloading the resulting audio.
   - Added `standardize_wav` to resample and mix downloaded wav segments to standard 16kHz mono format via FFmpeg.
   - Added `mix_tts_segments` to align and mix standardized WAV files into a single master voiceover file `tts_voice.wav` matching the video timeline using standard library `wave` and `struct`.

2. **AI Vocal Separation (`repurpose.py`)**:
   - Added `separate_vocals` using `demucs` command-line utility to extract the non-vocals background music (`no_vocals.wav`) track while eliminating original vocals.
   - Handled fallback gracefully by returning the original audio track if `demucs` is not installed or errors.

3. **FFmpeg Audio Ducking (`repurpose.py`)**:
   - Updated the `apply_subtitles_and_mask` multiplexing logic to apply sidechain compression (`sidechaincompress=threshold=0.1:ratio=4:attack=50:release=300`) to duck background music when the Vietnamese voiceover speaks, mixing them via `amix`.

4. **API and Web Dashboard UI (`app.py`)**:
   - Extended `run_background_pipeline` and `create_repurpose_job` endpoint to accept optional `siren_key` and `siren_voice` fields, defaulting to environment variables `SIREN_API_KEY` and `SIREN_VOICE_ID` if not provided.
   - Upgraded the HTML glassmorphic dashboard with input fields for the Siren API Key and Voice ID, passing them dynamically to background jobs.
